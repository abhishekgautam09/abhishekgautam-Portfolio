# Abhishek Gautam — Portfolio

Personal portfolio of **Abhishek Gautam**, a UX designer based in Bengaluru, India,
focused on internal, data-heavy, and workflow-driven products.

Built with **React + TypeScript + Vite + Tailwind CSS**.

## Development

```bash
npm install      # install dependencies
npm run dev      # start local dev server (http://localhost:5173)
npm run build    # production build into /dist
npm run preview  # preview the production build locally
```

## Notes

- The serif accent font (**PP Mondwest**) loads from `/public/PPMondwest-Regular.woff2`.
  Add that file to `public/` for the accent font to render.
- Project images currently use placeholder GIFs. Replace the `image` URLs in
  `src/components/ProjectsSection.tsx` and the marquee array in `src/App.tsx`
  with real screenshots of your work.
- Replace the parallax image in `src/components/TestimonialSection.tsx` with your own photo.
